Question-1: What are the different ways to select an element in the DOM?

We mainly use 5 ways to select an element in the DOM. They are:

  1. 1. By using getElementById("id")
  2. 2. By using getElementsByClassName("class")
  3. 3. By using getElementsByTagName("tag")
  4. 4. By using querySelector("css-selector")
  5. 5. By using querySelectorAll("css-selector")

Question-2: What is the difference between innerHTML, innerText, and textContent?

Differences between innerHTML, innerText, textContent:

  1. innerHTML: It returns the HTML content of an element (with hidden elements).
    (I have used it while creating an element to append to it's parent in this assignment.)
  2. innerText: It returns the text content of an element (without the texts of hidden elements).
    (I have used it while getting and setting the inner text of an element in this assignment.)
  3. textContent: It also returns the text content of an element (with the texts of hidden elements).
    (I haven't used it in this assignment.)

Question-3: What is event delegation in the DOM?

Event delegation is an efficient way to handle events if there are too many similar elements under the same parent node. In this case, we add an event lister to the parent node and try find out the target element. And then we execute the further procedure.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event, triggered by a child element, propagates through it's parents to the furthest parent that has an event listener. In this case, an event can get to the parent node, that was triggered by a child node. We can stop event bubbling by using the stopPropagation() function.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element we use createElement(). Example: const title = document.createElement("h1")
To add an element to a parent node we use appendChild(). Example: document.body.appendChild(title)
To remove an element we use remove(). Example: title.remove()
